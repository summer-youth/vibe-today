## 🎨 업데이트 내용

### 1️⃣ 로그인 페이지 개선
- ✅ 예쁜 로고와 일러스트 추가
- ✅ Google 로그인 버튼 디자인 개선

**특징:**

---

### 2️⃣ 홈 화면 - 일기 작성
- ✅ 오늘의 키워드 카드 (보라색 배경, 그림자)
- ✅ 일기 작성 텍스트 영역
- ✅ 글자 수 카운터 (500자 제한)
- ✅ "그림일기 만들기" 버튼
- ✅ AI 이미지 생성 로딩 (2초 후 랜덤 이미지 표시)
- ✅ 생성된 이미지 표시
- ✅ "새로 작성하기" 버튼

**흐름:**
1. 오늘의 키워드 확인 (예: #설렘)
2. 일기 작성 (최소 10자)
3. "그림일기 만들기" 버튼 클릭
4. 2초 로딩 후 AI 이미지 표시 (현재는 랜덤 이미지)
5. "새로 작성하기"로 다시 작성 가능

---

### 3️⃣ 기록 화면 - Grid 레이아웃
- ✅ 2열 Grid 레이아웃
- ✅ 예쁜 이미지 카드
- ✅ 각 카드에 키워드와 날짜 표시
- ✅ 그림자 효과
- ✅ 목 데이터 6개 표시
- ✅ Empty State (바이브가 없을 때)

**특징:**
- 정사각형 카드
- 이미지 위에 그라데이션 오버레이
- 키워드와 날짜가 하단에 표시
- 탭하면 상세 페이지로 이동 (추후 구현)

---

### 4️⃣ 프로필 화면
- ✅ 사용자 프로필 카드
- ✅ 통계 카드 (총 바이브, 연속 기록, 이번 달)
- ✅ 설정 메뉴 (알림, 데이터 내보내기, 앱 정보)
- ✅ 로그아웃 버튼
- ✅ 이모지 아이콘

---

### 5️⃣ 바텀 네비게이션
- ✅ 아이콘 추가 (이모지)
  - 홈: ✏️ (일기 작성)
  - 기록: 📚 (기록 보기)
  - 프로필: 👤 (프로필)
- ✅ 선택된 탭 강조
- ✅ 그림자 효과
- ✅ 깔끔한 디자인

## 🎨 디자인 시스템

### 색상
- **Primary**: `#6366F1` (보라색)
- **Secondary**: `#EC4899` (핑크색)
- **Background**: `#FAFBFF` (연한 파란색)
- **카드**: `#FFFFFF` (흰색)

### 타이포그래피
- **제목**: 28-36px, Bold (700-800)
- **본문**: 16-20px, Medium (500-600)
- **캡션**: 13-15px, Regular (400)

### 간격
- **여백**: 16-32px
- **카드 패딩**: 16-24px
- **둥근 모서리**: 12-20px

### 그림자
- 모든 카드에 일관된 그림자 적용
- 버튼에 Primary 색상 그림자
